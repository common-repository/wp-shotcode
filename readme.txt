== Description ==

ShotCode's Wordpress plugin allows you to easily integrate ShotCodes with your blog: just download the plugin zip file, unzip it into your wp-content/plugins/wp-shotcode and follow the configuration instructions under the Options -> ShotCode menu in the admin pages. Once configured, the plugin will automatically generate ShotCodes for each blog entry you write! 

== Installation ==

1. Upload `wp-shotcode.php` and `wp-shotcode_options_subpanel.inc` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the instructions under Options -> ShotCode on how to add the ShotCodes to your pages
